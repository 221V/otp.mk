synrc.mk
========

Makefile-based Erlang/OTP and reltool/relx compatible build solution that can use rebar also for deps/compilation.

Commands
--------

    make .applist
    make [compile]
    make console
    make start
    make attach
    make release

See real example of usage in https://github.com/5HT/skyline
